FoMI 2026 — 18 March 2026

AI-Powered Impact Analysis for Smart Manufacturing

Live presentation at FoMI 2026 by Tobias Lante. An AI agent answers: “What happens if machine SGM-004 goes down right now?” — using real factory data, Knowledge Graphs, and multi-agent discussion in under 10 seconds.

What you see

  • 1. A natural language question about a machine failure
  • 2. AI queries 5+ data sources via MCP tools (ERP, BDE, OEE, QMS, UNS)
  • 3. Knowledge Graph traversal finds downstream impact
  • 4. 4 specialist agents discuss from different perspectives
  • 5. Final recommendation with prioritized action plan

Tech Stack

  • LLM: Qwen 2.5 32B — local, llama.cpp, no cloud
  • Data: Factory Simulator v3 — 7 PostgreSQL databases
  • Knowledge Graph: Neo4j + 768d vector embeddings
  • Protocol: MCP (Model Context Protocol) for tool calling
  • OPC-UA: 35 machine servers, event-driven from DB
  • Infrastructure: Kubernetes, fully on-premise

About FoMI

FoMI (Forum for Manufacturing Intelligence) is an international platform for industrial AI, organized by DigitalFlowz. The 2026 edition: no slides, live demos only. This session demonstrated CESMII i3X principles (Interoperability, Integration, Intelligence) with open-source tools and local LLMs.
